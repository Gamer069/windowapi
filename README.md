# windowapi
This is a simple java window api to make making windows easy
#
Maven dependency:
```

  <artifactId>windowapi</artifactId>
  <version>(Latest version that is down below)</version>
</dependency>
```
, 
maven repository:
```
<repository>

</repository>  
```
